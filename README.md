# Monceri

Plataforma e-commerce para vender letreros neon personalizados y productos Monceri.
Dirigida a clientes finales y administradores de catalogo, pedidos e inventario.

## Stack
- TypeScript
- Next.js
- React
- Fastify
- Prisma
- PostgreSQL
- Refine
- Ant Design
- Tailwind CSS
- Three.js
- Zod
- Zustand
- pnpm

## Funcionalidades principales
- Storefront con catalogo, colecciones, carrito y checkout por WhatsApp.
- Configurador de letreros neon con calculo de precio.
- API modular para productos, categorias, colecciones, ordenes, cupones, inventario y uploads.
- Panel admin con autenticacion, dashboard y gestion operativa.
- Esquemas compartidos para validacion y contratos de datos.
